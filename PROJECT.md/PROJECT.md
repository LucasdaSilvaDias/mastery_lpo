# Mastery LPO

## Objetivo

O Mastery LPO é um PWA em React para atletas de levantamento de peso olímpico acompanharem treinos, evolução e cargas.

O sistema também terá um painel para o treinador criar, editar e publicar treinos para os atletas.

## Tipos de usuário

### Atleta

Pode:
- acessar o treino da semana;
- visualizar o treino do dia;
- registrar cargas utilizadas;
- marcar exercícios como concluídos;
- adicionar observações;
- acompanhar evolução;
- visualizar recordes pessoais.

### Treinador

Pode:
- cadastrar atletas;
- criar treinos;
- editar treinos;
- publicar treinos semanais;
- acompanhar registros dos atletas;
- visualizar evolução dos atletas.
